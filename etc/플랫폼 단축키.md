# 이클립스, 노션 단축키 모음

# 이클립스 단축키

> MacOS 기준

> 박재성(자바자기)님의 인프런 강의 - 만들어 가면서 배우는 자바 플레이그라운드에서 배운 내용 중, `"도구를 사용할때, 최대한 단축키를 이용해서 마우스 사용을 줄여라."`는 인상적인 멘트로 인해 단축키를 정리하고자한다.

> 플랫폼을 쓰면서 직접 사용하는 단축키를 실시간으로 업데이트하는 식으로 작성.

## 파일 관련 단축키

- `Cmd + n` 파일 생성
- `Cmd + fn + f11` 파일 실행

## 코드 작업 관련 단축키

- `Cmd + opt + 위 또는 아래키` 행 단위 복사

* `Cmd + Shift + L` 단축키 전체 목록보기
* `Cmd + D` 한줄 삭제
* `Cmd + opt + A` 멀티 칼럼 편집
* `Cmd + i` 자동 들여쓰기
* `Cmd + Shift + F` 자동 정렬
* `Cmd + /` 한줄 주석(토글)
* `/* ~ */` 여러줄 주석
* `opt + 위 or 아래` 행단위 이동
* `tab` 들여쓰기
* `tab + shift` 내어쓰기

# 노션 단축키

- `Shift + opt + L` - 블록링크 복사
