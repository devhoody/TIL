# 느낀점 (한줄~세줄)

- HTTP 강의 1회차 수강을 끝냈다. 노션에 정리한 내용을 하루에 한단원씩 보는 것으로 감을 잊지 않으려 한다. 내일부터 스프링 입문 강의를 수강을 시작해야겠다.

- 자바 객체지향 강의 1회차 수강을 끝냈다. 2회차 수강부터는 좀더 천천히 공부하면서 블로그에 정리하려한다. 진도는 계속해서 진행하려고한다.

- 하루 공부 계획을 세워보았다.

  - 자바 : 3시간
  - 스프링 : 2시간
  - SQL : 1시간
  - 실습 : 3시간
  - 코딩테스트 : 30분
  - 독서 : 30분
