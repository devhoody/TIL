# 배운 것

- 자바 객체지향 프로그래밍

  - 캡슐화
  - 메소드(인스턴스함수)
    - static method / instance method
  - 접근 제어 지시자
    - private / protected / public

- spring(모든 개발자를 위한 HTTP 웹 기본 지식)

  - HTTP
  - IP, 패킷(클라이언트 패킷, 서버 패킷)
  - ip 프로토콜의 한계 (비연결성, 비신뢰성, 프로그램 구분)
  - TCP, UDP
  - 인터넷 프로토콜 스택의 4계층
    - 애플리케이션 계층 - HTTP, FTP
    - 전송 계층 - TCP, UDP
    - 인터넷 계층 - IP
    - 네트워크 인터페이스 - LAN드라이버, LAN 장비
  - TCP 특징
    - TCP 3 way handshake
    - 데이터 전달 보증
    - 순서 보장
  - PORT

- 코딩테스트
  - 특수문자, 괄호, 백슬래시, 따옴표 출력하기
