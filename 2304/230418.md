# 느낀점 (한줄~세줄)

- 강의를 나누어 2시간씩 1,2,3교시로 나누어 듣기 시작했다. 각각 자바, SQL, Spring 수업으로 나누었으며,모든 강의를 들은 후 복습하고 정리까지 하루에 다하고자한다.

- 백준 오답노트에 있는 문제는 하루에 한번씩 풀어보자. 오늘 BufferReader, BufferedWriter를 배웠는데, 처리속도가 Scanner보다 빠르므로 주로 사용될 것이기에 익숙해지도록 해야겠다.

- 코딩테스트 문제 풀이 방식을 바꿔보았다.
  - 수정 전 : 문제풀이 후 내 문제풀이만 정리
  - 수정 후 : 문제풀이 후 내 문제풀이 + 남의 문제 풀이 연습 및 정리
    - 참고하기 좋은 블로그 : [Stranger's LAB](https://st-lab.tistory.com/287)
