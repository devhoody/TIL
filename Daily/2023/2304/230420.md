# 느낀점 (한줄~세줄)

- 자바 객체지향, HTTP 강의 1회차 수강이 거의 끝나간다. 2회차 수강부터는 보다 천천히 수강하면서 정보 체득에 중점을 두어야겠다. 내일부터 객체지향 1회차 수강을 끝내고 예외처리 강의를 수강하고자 한다.

- 내가 배운 내용의 용어들을 따로 정리한 나만의 사전을 만들생각이다. 그래서 헷갈리는 내용을 그떄그때 찾기 용이하게 할 생각이다.

- 내일부터 SQL 강의 수강을 시작하려고한다.
