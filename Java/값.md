# 값

## 값이란?

- 데이터의 특정한 표현 방식

## 진법

- 컴퓨터에는 2진법, 8진법, 10진법, 16진법이 있다.

```Java
int a = 0b1011 // 2진법 - 0b붙임
int b = 012 // 8진법 - 0붙임
Int c = 10; // 10진법
int d = 0x0a // 16진법 - 0x붙임
long e = 0x0aL // 16진법(8byte) - 0x, L붙임

// 1000단위로 언더바를 쓰면 수의 가독성을 늘릴 수 있다.
int x = 100_000_000;

```

# 정수형

- 정수값을 저장하는데 사용되는 데이터 유형
- 기본형은 int

## 정수형 값의 범위

- byte(1byte) : - 2^7 ~ 2^7 -1 (-128 ~ 127)
- short(2byte) : -2^15 ~ 2^15 -1
- `int(4byte)` : -2^31 ~ 2^31 -1 (약 20억)
- long(8byte) : -2^63 ~ 2^63 -1

# 실수형

- 소수점 이하까지의 숫자를 포함하는 수를 저장하는데 사용되는 데이터 유형

- float : 4byte, double(기본형) : 8byte

## 실수형의 정밀도

- float : 7자리
- `double` : 15자리

## 고정 소수점 방식, 부동 소수점 방식

### 고점 소수점 방식 (fixed point)

- 부호비트(1bit) + 정수부(15bit) + 소수부(16bit)

# 문자형

- 단일 문자를 표현하는데 사용되는 데이터 유형
- `작은따옴표(' ')`로 표현하며 String은 기본형이 아님에 주의하자.
- 문자열의 경우 'H''E''L''L''O'`'O'`로 `O`가 붙는다면 문자열 표기가 가능하다.
- char 1byte

# 논리형

- True나 False값을 표현하는데 사용되는 데이터 유형
- 조건문의 참 또는 거짓을 판단하는데 주로 사용된다.
- boolean ??

# 값의 변환

- Byte가 큰 자료형에 상대적으로 작은 자료형은 변경이 가능하다.
- ex) `long A = 10; 가능`, `int B = 10L; 불가능` 컴파일 에러 발생
